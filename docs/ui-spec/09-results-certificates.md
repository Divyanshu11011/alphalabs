## 🧱 COMPONENT BREAKDOWN

### Result Card (List Item)

```
Component: ResultCard
├── Status Indicator
│   ├── 🟢 Profitable
│   └── 🔴 Loss
│
├── Header
│   ├── Type Badge: "BACKTEST" or "FORWARD TEST"
│   ├── Test Number: "#27"
│   ├── Live Badge (if forward): "[LIVE]"
│   └── PnL: "+23.4%" (color coded)
│
├── Meta
│   ├── Agent + Asset + Mode
│   ├── Date + Duration + Trade count
│   └── Win Rate + Max DD
│
├── Divider
│
└── Actions
    ├── [View Details] → Always visible
    ├── [Certificate] → Only if profitable
    └── [Share] → Only if profitable
```

### Scorecard Component

```
Component: ResultScorecard
├── Primary Metrics Row
│   ├── Total PnL ($ and %)
│   ├── Win Rate
│   ├── Profit Factor
│   └── Sharpe Ratio
│
├── Secondary Metrics Row
│   ├── Max Drawdown
│   ├── Average Trade
│   ├── Best Trade
│   └── Worst Trade
│
└── Styling
    ├── Large numbers
    ├── Color coding (green/red)
    └── Subtle icons
```

### Certificate Generator

```
Component: CertificateGenerator
├── Preview
│   ├── Rendered certificate image
│   └── Zoom/Pan capabilities
│
├── Customization (optional)
│   ├── Include reasoning trace: Yes/No
│   ├── Include trade list: Yes/No
│   └── Theme: Dark/Light
│
├── Export Options
│   ├── Download PDF
│   ├── Download PNG
│   ├── Download Full Report (PDF with all data)
│   └── Share Link
│
└── Social Sharing
    ├── Share on X/Twitter
    ├── Copy share link
    └── QR code for verification
```
