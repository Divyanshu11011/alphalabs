# AlphaLab UI/UX Specification
## Part 10: Settings Screens

---

## 📍 SCREEN OVERVIEW

**Routes:** 
- `/dashboard/settings` - General settings
- `/dashboard/settings/api-keys` - API key management
- `/dashboard/settings/preferences` - Trading preferences

**Purpose:** Manage account settings, API keys, and default trading configurations.
